Mothership an open-source platform-as-service for small applications. Designed to be used for deploying internal-facing apps for companies, Mothership is easy to setup. Companies can have their own self-hosted PaaS with a simple mothership-setup command.

Once set up, users can use mothership command-line interface to web interface to deploy their apps to the cloud without any need for knowledge of servers or containers.
